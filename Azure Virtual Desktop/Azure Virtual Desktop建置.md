# Azure Virtual Desktop建置
(一) 新增主要網域
1. 已全域管理員身分帳號登入Azure Portal
2. 進入Azure Active Directory->自訂網域名稱->新增自訂網域
3. 前往該網域的DNS新增一筆TXT或者MX紀錄
4. 驗證網域 <br>
